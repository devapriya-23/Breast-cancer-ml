# Breast Cancer Prediction using Machine Learning

This project predicts whether a breast tumor is *Malignant* or *Benign* using Machine Learning techniques.

## Dataset
Breast Cancer Wisconsin dataset.

Target variable:
- 0 → Benign  
- 1 → Malignant  

## Algorithm Used
- Logistic Regression

## Steps Involved
1. Data Loading
2. Data Cleaning (dropping unnecessary columns)
3. Feature selection
4. Train-test split
5. Model training
6. Prediction
7. Model evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix
- ROC AUC Score

## Results
Accuracy achieved: *~98%*  
ROC AUC Score: *~0.97*

## Interpretation
- Positive coefficient increases cancer probability.
- Negative coefficient decreases cancer probability.
- Higher absolute value indicates more important feature.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Conclusion
This project demonstrates how Machine Learning can assist in early breast cancer detection using classification techniques.
